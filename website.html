Entendo Comunicação Cliente x Server

Esse módulo do curso foi dividido em algumas aulas divididas em etapas. Cada etapa trás informações importantes para o aprendizado do aluno, e os ensinamentos passados estarão descritos logo abaixo.

1 História da Web

Na etapa "História da Web", como o próprio nome sugere, os alunos são introduzidos ao professor do curso e começa a ter início a trilha com o HTML, é nessa etapa que os alunos têm acesso a informações interessantes tais como informações acerca dos primeiros computadores e o respectivo funcionamento de cada um. Além disso, é descrito o surgimento da Internet e da Web, deixando frisado os nomes importantes de todos os processos apresentados.

2 O que são Clients

A etapa nomeada "O que são Clients", apresenta justamente onde os alunos vão entender do que se trata o lado client da aplicação web, que nada mais é do que o lado que vai estar consumindo as informações dadas. Também é descrito o uso e a importância do Cachê, demostrando como ele é utilizado na navegação web.

3 O que são Servers

Após apresentados ao lado Client e compreendido seus conceitos, os alunos são agora apresentado ao conceito Server. Os servidores, Servers, nada mais é que o lado da aplicação Web que fornecerá as informações para o cliente, utilizando a conexão com a internet para realizar a conexão entre eles. Por tanto, serve de apoio para as demais máquinas da rede.

4 Linguagens de Programação

Na última etapa, após apresentados a conceitos importantes no quetange navegação Web, houve a apresentação das Linguagens de Programação que no curso foram descritas como "uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programa (software). Ademais, foi descrito as linguagens de programação mais utilizadas no mercado.


Introdução ao HTML na prática

Esse módulo do curso, diferentemente do anterior, não teve suas aulas divididas em módulos, já que o enfoque agora, passa de ser a parte teórica, e se torna a prática. As aulas desse módulo são justamente isso, é apresentado na prática as tags do HTML, tendo seu uso demonstrado em aula para um melhor entendimento. Aqui foi apresentadas tags como parágrafo, a diferenciação dos títulos e além das tags também foi demonstrado uma estrturação de página, encerrando assim os Primeiros Passos com HTML do curso Formação HTML Web Developer.
